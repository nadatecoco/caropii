AI活用型 健康管理アプリ

📱 現在のアプリ状況

✅ 実装済み機能

  • 食事記録システム：食材選択による簡単な食事記録
  • 栄養可視化：PFC（タンパク質・脂質・炭水化物）を横棒グラフで表示
  • AI栄養分析：Google Gemini APIで今日の食事を分析してアドバイス
  • Rails API連携：iOSアプリ ⇔ バックエンド間でデータ送受信

🛠 技術構成

フロントエンド
SwiftUI (iOS)
バックエンド
Ruby on Rails 8.0
AI分析
Google Gemini API
データベース
SQLite

🎯 アプリの流れ

食材選択 → 栄養グラフ表示 → AI分析依頼 → 改善アドバイス取得

🚀 今後追加予定の機能

🎙️ 音声プロンプト入力

  • 「今日は鮭おにぎり2個とサラダチキン食べました」みたいに自然言語で記録
  • LLMが食材を自動識別・抽出

🤖 未知食材のAI栄養価算出

  • データベースにない食材もAIが栄養価を推定
  • ユーザー確認を経てデータベースに自動追加

🍎 Apple Watch連携

  • 睡眠データ：睡眠時間・質を自動取得
  • ワークアウトデータ：心拍数・消費カロリー・運動時間
  • 3軸統合分析：睡眠 + 運動 + 食事で総合的な健康アドバイス

💭 開発で工夫したポイント

  • シンプルなUX：タップだけで簡単に食事記録
  • リアルタイム可視化：Swift Chartsで栄養バランスを即座にグラフ表示
  • AI活用：単純な記録ではなく、実用的なアドバイスを提供
  • 実装優先：複雑な設計より、まず動くものを作成

🎯 最終的な目標

「記録するだけ」で終わらない健康管理アプリ
睡眠・運動・食事の3つのデータを統合し、
AIが個人に最適化された健康アドバイスを提供する
